pydumpfs

Description

pydumpfs is a backup tool using hard links. If a file is unchanged, pydumpfs makes a hard link to previous one. So you don't waste disk space.

pydumpfs also removes old backups which are more than 93 days old.

Requirements

  • FreeBSD 8.2
  • Python 2.6.6

Download

  • http://github.com/SumiTomohiko/pydumpfs

Usage

pydumpfs dest_dir src_dir...

pydumpfs makes a directory under dest_dir. Its name is in the form of yyyy-mm-dd_hour:minute:second.millisecond. You can give more than one src_dir.
